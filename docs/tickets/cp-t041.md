## Problem Statement

The technical_writer flagged `docs/guides/memory-explorer.md` as unreviewed during the Phase 2 docs pass (2026-03-20). ## Scope

1. **Port audit** — Find and correct any references to port 4000. Correct values: server port 3002, Vite dev port 5173.

2. **CP-T036 coverage** — Add documentation for:
   - Entity Detail view at `/memory/:entityType/:entityId`
   - Temporal History view at `/memory/:entityType/:entityId/:key`
   - How to navigate to entity detail from a Memory Explorer fact row
   - The "entity field is null" known limitation (entities table not in current schema)
   - How to read the temporal history intervals and supersession chain

3. **Accuracy pass** — Verify the guide accurately reflects the current Memory Explorer filter capabilities (entityType, entityId, key, source, createdBy, minConfidence, search) and does not reference any Phase 2 features as if they are Phase 1.

4. **Cross-reference** — Add a pointer to `docs/reference/known-issues.md` (CP-T040) for known Memory Explorer limitations (search substring-only, entity null field).
